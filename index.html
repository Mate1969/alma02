Az alma

The logo for Apple Inc is a Private Use Area (PUA) character that is supported on iOS and macOS. This character is not recommended for interchange as it is only intended for support on Apple platforms. As of August 2021, the Twitter website also displays this character as the Apple logo, in addition to another PUA character which displays the Twitter logo.
